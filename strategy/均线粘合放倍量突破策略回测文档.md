# 均线粘合放倍量突破策略回测文档

## 1. 策略名称

**均线粘合放倍量突破策略**

简称：`MA-Convergence Volume Breakout`

策略类型：

- A股日线选股策略
- 趋势突破策略
- 均线粘合突破策略
- 量价共振策略

---

## 2. 策略核心思想

本策略试图捕捉以下形态：

1. 个股处于中长期上升趋势中；
2. 短期均线 MA5、MA10、MA20 高度粘合；
3. 股价向上突破粘合均线区域；
4. 当日成交量较前一日明显放大，形成放倍量突破；
5. 认为该类形态可能代表资金重新介入，股价进入短线或波段启动阶段。

策略本质是：

> 在中长期趋势向上的股票中，寻找短期均线压缩后的放量向上突破买点。

---

## 3. 原始通达信公式

```text
价五:=MA(CLOSE,5);
价十:=MA(CLOSE,10);
价二十:=MA(CLOSE,20);
价六十:=MA(CLOSE,60);
价一二零:=MA(CLOSE,120);
价二四零:=MA(CLOSE,240);

{一、趋势过滤：有MA240看MA240，没有MA240看MA120或MA60}
BC:=BARSCOUNT(CLOSE);

有240:=BC>=260;
有120:=BC>=140;
有60:=BC>=80;

MA240UP:=IF(有240,价二四零>REF(价二四零,5) AND 价二四零>REF(价二四零,20),0);
MA120UP:=IF(有120,价一二零>REF(价一二零,5) AND 价一二零>REF(价一二零,20),0);
MA60UP:=IF(有60,价六十>REF(价六十,5) AND 价六十>REF(价六十,20),0);

趋势OK:=IF(有240,MA240UP,MA120UP OR MA60UP);

{二、均线粘合阈值}
NHLIMIT:=2.5;

{三、当前MA5/MA10/MA20粘合}
CURMAX:=MAX(MAX(价五,价十),价二十);
CURMIN:=MIN(MIN(价五,价十),价二十);
CURNHD:=(CURMAX-CURMIN)/CURMIN*100;
CURNH:=CURNHD<=NHLIMIT;

当前粘合:=CURNH AND CLOSE>CURMAX;

{四、扣抵后的MA5/MA10/MA20粘合}
KDM5:=(SUM(CLOSE,5)-REF(CLOSE,4)+CLOSE)/5;
KDM10:=(SUM(CLOSE,10)-REF(CLOSE,9)+CLOSE)/10;
KDM20:=(SUM(CLOSE,20)-REF(CLOSE,19)+CLOSE)/20;

KDMAX:=MAX(MAX(KDM5,KDM10),KDM20);
KDMIN:=MIN(MIN(KDM5,KDM10),KDM20);
KDNHD:=(KDMAX-KDMIN)/KDMIN*100;
KDNH:=KDNHD<=NHLIMIT;

扣抵粘合:=KDNH AND CLOSE>KDMAX;

{五、均线信号：当前粘合 或 扣抵粘合}
均线信号:=趋势OK AND (当前粘合 OR 扣抵粘合);

{六、放倍量条件}
倍量:=VOL>=2*REF(VOL,1);

{八、最终选股}
XG:均线信号 AND 倍量;
```

## 4. 策略逻辑拆解

### 4.1 均线定义

对每只股票，在交易日 `t` 计算：

```text
MA5(t)   = 最近 5 个交易日收盘价均值
MA10(t)  = 最近 10 个交易日收盘价均值
MA20(t)  = 最近 20 个交易日收盘价均值
MA60(t)  = 最近 60 个交易日收盘价均值
MA120(t) = 最近 120 个交易日收盘价均值
MA240(t) = 最近 240 个交易日收盘价均值
```

---

## 5. 趋势过滤条件

### 5.1 上市交易天数判断

```text
BC = 当前股票已有日线K线数量
```

判断是否具备足够长周期均线：

```text
有240 = BC >= 260
有120 = BC >= 140
有60  = BC >= 80
```

这里使用 `260 / 140 / 80`，而不是直接使用 `240 / 120 / 60`，是为了保证均线本身以及回看斜率判断有足够数据。

---

### 5.2 长周期趋势向上

如果股票有 MA240，则优先使用 MA240 判断趋势：

```text
MA240UP = MA240(t) > MA240(t-5) AND MA240(t) > MA240(t-20)
```

如果没有足够 MA240 数据，则使用 MA120 或 MA60：

```text
MA120UP = MA120(t) > MA120(t-5) AND MA120(t) > MA120(t-20)
MA60UP  = MA60(t)  > MA60(t-5)  AND MA60(t)  > MA60(t-20)
```

最终趋势过滤：

```text
如果 有240:
    趋势OK = MA240UP
否则:
    趋势OK = MA120UP OR MA60UP
```

含义：

- 老股票优先看 MA240 是否向上；
- 次新股没有 MA240 时，用 MA120 或 MA60 代替；
- 趋势向下或长期均线走平的股票不进入候选池。

---

## 6. 均线粘合条件

### 6.1 粘合阈值

```text
NHLIMIT = 2.5
```

即 MA5、MA10、MA20 三条均线之间的最大偏离不超过 2.5%。

---

### 6.2 当前均线粘合

计算 MA5、MA10、MA20 中的最大值和最小值：

```text
CURMAX = max(MA5, MA10, MA20)
CURMIN = min(MA5, MA10, MA20)
```

计算当前均线粘合度：

```text
CURNHD = (CURMAX - CURMIN) / CURMIN * 100
```

当前粘合条件：

```text
CURNH = CURNHD <= 2.5
```

当前突破条件：

```text
当前粘合 = CURNH AND Close(t) > CURMAX
```

含义：

- MA5、MA10、MA20 已经高度接近；
- 当日收盘价站上三条短期均线的最高值；
- 股价从短期均线粘合区域向上突破。

---

## 7. 扣抵均线粘合条件

### 7.1 扣抵均线定义

原公式中：

```text
KDM5  = (SUM(CLOSE,5)  - REF(CLOSE,4)  + CLOSE) / 5
KDM10 = (SUM(CLOSE,10) - REF(CLOSE,9)  + CLOSE) / 10
KDM20 = (SUM(CLOSE,20) - REF(CLOSE,19) + CLOSE) / 20
```

其含义可以理解为：

> 假设下一个交易日收盘价仍然等于今天收盘价，提前计算下一日 MA5、MA10、MA20 的理论位置。

例如：

```text
KDM5(t) = [Close(t) + Close(t) + Close(t-1) + Close(t-2) + Close(t-3)] / 5
```

本质上是把 MA5 中即将被扣掉的老价格替换为当前价格。

该计算只使用当前及历史数据，不使用未来价格。

---

### 7.2 扣抵均线粘合判断

```text
KDMAX = max(KDM5, KDM10, KDM20)
KDMIN = min(KDM5, KDM10, KDM20)
KDNHD = (KDMAX - KDMIN) / KDMIN * 100
KDNH  = KDNHD <= 2.5
```

扣抵粘合突破：

```text
扣抵粘合 = KDNH AND Close(t) > KDMAX
```

含义：

- 当前均线可能还没有完全粘合；
- 但按扣抵价格推演，下一日 MA5、MA10、MA20 可能即将粘合；
- 股价已经提前站上扣抵后的均线区域；
- 该条件用于提前捕捉即将形成的粘合突破形态。

---

## 8. 均线信号

```text
均线信号 = 趋势OK AND (当前粘合 OR 扣抵粘合)
```

即：

1. 中长期趋势向上；
2. 当前 MA5、MA10、MA20 粘合并突破；

或者：

1. 中长期趋势向上；
2. 扣抵后的 MA5、MA10、MA20 即将粘合并突破。

---

## 9. 放倍量条件

```text
倍量 = Volume(t) >= 2 * Volume(t-1)
```

含义：

- 当日成交量至少是前一交易日的 2 倍；
- 说明突破不是无量突破；
- 用成交量确认资金介入强度。

---

## 10. 最终选股条件

```text
XG = 均线信号 AND 倍量
```

即：

```text
XG =
    趋势OK
    AND
    (
        当前粘合
        OR
        扣抵粘合
    )
    AND
    当日成交量 >= 前一日成交量的 2 倍
```

---

## 11. 回测执行原则

由于本策略使用了当日收盘价和当日成交量，因此在日线回测中不能假设当天开盘买入。

默认回测方式：

```text
T日收盘后计算信号
T+1日开盘买入
```

这样可以避免盘后数据提前使用的问题。

---

## 12. 股票池定义

### 12.1 基础股票池

默认股票池：

```text
全部A股
```

可选增强过滤：

```text
剔除停牌股
剔除退市整理股
剔除上市不足 80 个交易日的股票
剔除当日一字涨停无法买入的股票
剔除当日一字跌停无法卖出的股票
```

### 12.2 ST 股票处理

原始公式没有剔除 ST 股票，因此为了完全复现公式，默认：

```text
FILTER_ST = False
```

如果用于实盘或稳健回测，建议增加：

```text
FILTER_ST = True
```

原因：

- ST 股票涨跌停限制不同；
- 基本面风险较高；
- 流动性和退市风险更大；
- 容易扭曲策略回测结果。

## 13. 调仓规则

### 13.1 信号生成时间

```text
每个交易日收盘后，根据当日日线数据计算 XG 信号。
```

### 13.2 买入时间

```text
T日出现 XG 信号；
T+1日开盘买入。
```

### 13.3 买入价格

默认：

```text
买入价 = T+1日开盘价 * (1 + 滑点)
```

建议滑点参数：

```text
SLIPPAGE = 0.001
```

即单边 0.1%。

---

## 14. 持仓方式

### 14.1 等权持仓

所有入选标的平均持仓：

```text
单只股票目标权重 = 1 / 入选股票数量
```

例如当日选出 26 只股票：

```text
每只股票目标权重 = 1 / 26 ≈ 3.846%
```

组合收益率计算：

```text
组合收益率 = 所有持仓股票收益率的算术平均值
```

### 14.2 最大持仓数量

为了避免持仓过于分散，建议设置最大持仓数量：

```text
MAX_HOLD_NUM = 30
```

如果当日信号数量小于等于 30：

```text
全部买入
```

如果当日信号数量大于 30，则需要排序筛选。

---

## 15. 候选股排序规则

原始公式没有排序逻辑。

如果本地回测系统必须限制持仓数量，可以使用以下排序方式。

### 15.1 推荐排序因子

#### 1. 均线粘合度越低越好

```text
粘合度 = min(CURNHD, KDNHD)
```

排序：

```text
粘合度升序
```

含义：MA5、MA10、MA20 越接近，形态压缩越充分。

#### 2. 突破幅度适中

```text
突破幅度 = Close / max(CURMAX, KDMAX) - 1
```

建议保留：

```text
0 < 突破幅度 <= 0.08
```

避免过度追高。

#### 3. 成交量放大强度

```text
量比 = Volume(t) / Volume(t-1)
```

排序：

```text
量比降序
```

但需要避免极端异常放量。

建议过滤：

```text
量比 <= 8
```

#### 4. 流动性

```text
成交额 = Close * Volume
```

建议过滤：

```text
近20日平均成交额 >= 3000万
```

如果做小微盘策略，可以降低至：

```text
近20日平均成交额 >= 1000万
```

## 16. 默认卖出规则

原始选股公式只定义了买入信号，没有定义卖出规则。

为了使策略可回测，需要明确卖出逻辑。建议先使用最小假设版本，评估买点本身是否有效。

### 16.1 版本A：固定持有期卖出

这是最适合验证公式有效性的版本。

```text
T日收盘出现信号；
T+1日开盘买入；
持有 N 个交易日；
第 N+1 个交易日开盘卖出。
```

默认参数：

```text
HOLD_DAYS = 5
```

可测试参数：

```text
HOLD_DAYS = 1
HOLD_DAYS = 3
HOLD_DAYS = 5
HOLD_DAYS = 10
HOLD_DAYS = 20
```

用途：

- 检验信号后 1 / 3 / 5 / 10 / 20 日的平均收益；
- 判断该公式更适合超短、短线还是波段。

### 16.2 版本B：跌破 MA20 卖出

适合波段跟踪。

卖出条件：

```text
Close(t) < MA20(t)
```

执行：

```text
T日收盘触发卖出信号；
T+1日开盘卖出。
```

优点：

- 能持有趋势延续的股票；
- 不会因为固定时间过早卖出。

缺点：

- 回撤可能加大；
- 假突破后止损不够快。

### 16.3 版本C：固定持有期 + 止损

推荐作为实盘模拟版本。

买入后同时设置：

```text
最大持有期 = 10个交易日
止损线 = -5%
跌破 MA20 卖出
```

卖出条件满足任一即可：

```text
1. 持有满 10 个交易日；
2. 当前收益率 <= -5%；
3. Close(t) < MA20(t)；
4. 股票停牌、退市风险或无法交易。
```

默认参数：

```text
MAX_HOLD_DAYS = 10
STOP_LOSS = -0.05
```

## 19. Python 风格伪代码

```python
import numpy as np
import pandas as pd


def calc_signal(df: pd.DataFrame) -> pd.DataFrame:
    """
    df: 单只股票日线数据
    columns: date, open, high, low, close, volume, amount
    """

    df = df.copy()

    df["ma5"] = df["close"].rolling(5).mean()
    df["ma10"] = df["close"].rolling(10).mean()
    df["ma20"] = df["close"].rolling(20).mean()
    df["ma60"] = df["close"].rolling(60).mean()
    df["ma120"] = df["close"].rolling(120).mean()
    df["ma240"] = df["close"].rolling(240).mean()

    df["bar_count"] = np.arange(1, len(df) + 1)

    df["has_240"] = df["bar_count"] >= 260
    df["has_120"] = df["bar_count"] >= 140
    df["has_60"] = df["bar_count"] >= 80

    df["ma240_up"] = (
        df["has_240"]
        & (df["ma240"] > df["ma240"].shift(5))
        & (df["ma240"] > df["ma240"].shift(20))
    )

    df["ma120_up"] = (
        df["has_120"]
        & (df["ma120"] > df["ma120"].shift(5))
        & (df["ma120"] > df["ma120"].shift(20))
    )

    df["ma60_up"] = (
        df["has_60"]
        & (df["ma60"] > df["ma60"].shift(5))
        & (df["ma60"] > df["ma60"].shift(20))
    )

    df["trend_ok"] = np.where(
        df["has_240"],
        df["ma240_up"],
        df["ma120_up"] | df["ma60_up"]
    ).astype(bool)

    curmax = df[["ma5", "ma10", "ma20"]].max(axis=1)
    curmin = df[["ma5", "ma10", "ma20"]].min(axis=1)

    df["cur_nhd"] = (curmax - curmin) / curmin * 100
    df["cur_nh"] = df["cur_nhd"] <= 2.5

    df["current_converge_break"] = df["cur_nh"] & (df["close"] > curmax)

    df["kdm5"] = (
        df["close"].rolling(5).sum()
        - df["close"].shift(4)
        + df["close"]
    ) / 5

    df["kdm10"] = (
        df["close"].rolling(10).sum()
        - df["close"].shift(9)
        + df["close"]
    ) / 10

    df["kdm20"] = (
        df["close"].rolling(20).sum()
        - df["close"].shift(19)
        + df["close"]
    ) / 20

    kdmax = df[["kdm5", "kdm10", "kdm20"]].max(axis=1)
    kdmin = df[["kdm5", "kdm10", "kdm20"]].min(axis=1)

    df["kd_nhd"] = (kdmax - kdmin) / kdmin * 100
    df["kd_nh"] = df["kd_nhd"] <= 2.5

    df["deduct_converge_break"] = df["kd_nh"] & (df["close"] > kdmax)

    df["ma_signal"] = (
        df["trend_ok"]
        & (df["current_converge_break"] | df["deduct_converge_break"])
    )

    df["volume_signal"] = df["volume"] >= 2 * df["volume"].shift(1)

    df["xg"] = df["ma_signal"] & df["volume_signal"]

    return df
```

## 21. 关键注意事项

### 21.1 不能在信号日开盘买入

因为策略条件使用了：

```text
Close(t)
Volume(t)
```

这两个数据只有在收盘后才完整。

所以严谨回测必须使用：

```text
T日收盘出信号
T+1日开盘买入
```

否则会产生盘后数据提前使用的问题。

### 21.2 扣抵均线不是未来函数

扣抵均线使用的是：

```text
历史收盘价 + 当前收盘价
```

并没有使用未来价格。

它只是模拟：

> 如果下一日价格不变，下一日均线会如何变化。

因此可以在 T 日收盘后计算。

### 21.3 需要单独测试不同持有期

该公式本身是买点公式，不是完整交易系统。

建议重点测试：

```text
信号后1日收益
信号后3日收益
信号后5日收益
信号后10日收益
信号后20日收益
```

如果 1 至 5 日收益较好，说明偏短线。

如果 10 至 20 日收益更好，说明可以扩展为波段策略。

## 23. 策略优点

1. 逻辑清晰，完全基于量价和均线结构；
2. 不依赖财务数据，适合每日快速扫描；
3. 对启动形态较敏感，容易捕捉放量突破；
4. 在市场弱势时，如果仍能出现正收益，说明选股条件具备一定强度；
5. 可解释性强，适合人工复盘和本地量化系统落地。

## 24. 策略风险

### 24.1 容易追高

放倍量突破当日可能已经涨幅较大，T+1 开盘买入可能面临高开回落。

需要测试：

```text
信号日涨幅分组收益
T+1高开幅度分组收益
```

建议增加过滤：

```text
信号日涨幅 <= 8%
T+1开盘涨幅 <= 5%
```

### 24.2 假突破风险

均线粘合后的放量突破可能是假突破。

典型表现：

```text
当日放量突破
次日冲高回落
随后跌回 MA20
```

应通过止损或跌破 MA20 卖出控制。

### 24.3 ST 和低流动性股票干扰

原公式会选出 ST 股票。

如果不剔除 ST，回测可能受到以下影响：

```text
涨跌停制度不同
流动性不足
退市风险
连续一字板不可成交
```

建议分别回测：

```text
包含ST版本
剔除ST版本
```

### 24.4 市场环境依赖

该策略更适合以下环境：

```text
指数震荡偏强
题材活跃
成交额放大
中小盘活跃
```

不适合：

```text
系统性下跌
缩量阴跌
高位退潮
极端流动性收缩
```

## 25. 建议扩展方向

### 25.1 增加市场环境过滤

可以增加指数过滤：

```text
中证1000 > MA20
或
万得全A > MA20
或
全市场上涨家数 > 下跌家数
```

避免在系统性下跌期间频繁买入。

### 25.2 增加信号日涨幅过滤

建议测试：

```text
信号日涨幅 <= 6%
信号日涨幅 <= 8%
信号日涨幅 <= 10%
```

如果涨幅过大，次日追入性价比可能下降。

### 25.3 增加成交额过滤

建议测试：

```text
近20日平均成交额 >= 1000万
近20日平均成交额 >= 3000万
近20日平均成交额 >= 5000万
```

不同资金规模应使用不同流动性门槛。

### 25.4 增加题材强度过滤

可以统计股票所属行业或概念板块强度。

例如：

```text
所属行业近5日涨幅排名进入前30%
所属行业近20日涨幅排名进入前30%
所属概念板块出现涨停潮
```

该策略本质是形态突破，如果叠加题材强度，可能提高成功率。

## 27. 最小可用策略定义

如果只实现一个最小版本，建议使用下面规则：

```text
每日收盘后计算 XG 信号；
次日开盘买入所有入选股票；
所有股票等权持仓；
固定持有 5 个交易日；
第 6 个交易日开盘卖出；
手续费万三；
卖出印花税千一；
单边滑点 0.1%；
不剔除 ST，用于完全复现公式；
之后再测试剔除 ST 后的版本。
```

## 28. 策略结论

该公式是一个典型的：

```text
中长期趋势向上
+
短期均线粘合
+
放倍量突破
```

的选股模型。

它不是完整交易系统，而是一个买点筛选器。

本地量化系统落地时，必须补充：

```text
买入时点
卖出规则
持仓数量
等权规则
流动性过滤
交易成本
涨跌停处理
ST处理
```

建议先以固定持有期方式验证信号有效性，再决定是否扩展为波段持仓模型。

最优先测试的不是年化收益，而是：

```text
信号后 1 / 3 / 5 / 10 / 20 日收益分布
```

如果这些收益分布呈现稳定正偏，则说明该均线粘合放量突破信号具备进一步策略化价值。
